삼항 조건 연산자

삼항 조건 연산자는 조건식(true 또는 flase의 결괏값을 반환)의 데이터 결과에 따라 실행 결과가 달라집니다. 삼항 조건 연산자는 연산하기 위해서 피연산자가 3개 필요한 삼항 연산자입니다.

삼항 조건 연산자는 연산한 결과 조건식의 만족 여부에 따라 실행하는 스크립트의 코드를 다르게 실행하고자 할 때 사용합니다.

기본형) 조건식?실행문1:실행문2;

예시) 입력받은 나잇값이 20세 이상이면 '성인'이라 출력되게 하고, 미만일 경우 '미성년자'라고 출력하고자 할 때 사용
