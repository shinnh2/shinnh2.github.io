제어문

제어문의 종류: 조건문, 선택문, 반복문

조건문: if, else, else if

다음 값은 조건식에 false를 반환합니다: 0, null, ""(빈 문자), undefined

선택문: switch

반복문: while, do while, for

  • while: 특정한 조건을 만족할 때가지 실행문을 반복하여 실행한다.
  • do while: 조건에 상관없이 최소 한 번 실행문을 실행하고 조건식을 검사한다.
  • for: 조건식을 만족할 때까지 특정 실행문을 반복하여 실행하되, 몇 번 실행하는지 정할 수 있다.
  • break: 반복문에서 실행하면 조건식과 상관없이 강제로 반복문을 종료시킨다.(원하는 부분 다음부터 출력하지 않을 수 있다.)
  • continue: 반복문에서만 사용가능하며(원하는 부분만 선택적으로 출력할 수 있다.),
    while문의 경우 continue다음의 실행문을 무시하고 바로 조건식으로 이동해 조건 검사를 수행한다.
    for문의 경우 continue 다음의 실행문을 무시하고 바로 증감식으로 이동해 증감 연산을 실행한다.
while 기본형
var 변수=초깃값;
while(조건식){
  실행문;
  증감식;
}
do while 기본형
var 변수=초깃값;
do{
  실행문;
  증감식;
}while(조건식)
for문 기본형
for(초깃값; 조건식; 증감식){
  실행문;
}
break문 기본형 - for문에서
for(초깃값; 조건식; 증감식){
  break; //반복문을 강제로 종료합니다.
  실행문;
}
break문 기본형 - while문에서
var 변수=초깃값;
while(조건식){
  break; //반복문을 강제로 종료합니다.
실행문;
증감식;
}
continue문 기본형 - for문에서
for(초깃값; 조건식; 증감식){
continue; //다음에 오는 실행문은 무시하고 바로 증감식으로 이동
실행문;
}
continue문 기본형 - while문에서
var 변수=초깃값;
while(조건식){
  증감식;
  continue; //다음에 오는 실행문은 무시하고 바로 조건식으로 이동
실행문;
}
