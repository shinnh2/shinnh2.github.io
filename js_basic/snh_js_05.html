함수

데이터만 저장할 수 있는 변수와 달리 실행문을 저장할 수 있음

함수를 호출해야 실행문이 실행된다.

  • 함수 기본
  • 내장 함수
  • return문
  • 지역 변수와 전역 변수

1. 함수 기본

함수 기본형1-함수 정의문
function 함수명(){
  실행문;
}
함수 기본형2-익명적 함수
참조 변수=function (){
  실행문;
}
함수 실행 기본형
함수명();
또는 참조 변수();

함수 정의와 달리 매개 변수가 있는 함수는 변수를 받아서 함수를 실행할 수 있다.

매개 함수 정의 기본형
function 함수명(매개 변수1, 매개 변수2, ...매개 변수n){
  스크립트 실행문;
}
매개 함수 실행 기본형
function 함수명(데이터1, 데이터2, ...데이터n);

2. 내장 함수

함수 호출만으로 함수 사용 가능

내장 함수의 종류
종류설명사용 예
parseInt()문자형 데이터를 정수형 데이터로 바꿉니다.parseInt("5.12")→5
parseFloat()문자형 데이터를 실수형 데이터로 바꿉니다.parseFloat("5.12")→5.12
String()문자형 데이터로 바꿉니다.String(5)→"5"
Number()숫자형 데이터로 바꿉니다.Number("5")→5
Boolean()논리형 데이터로 바꿉니다.Boolean(5)→true
isNaN()문자형 데이터를 정수형 데이터로 바꿉니다.isNaN("5-3")→true
eval()문자형 데이터를 큰따옴표가 없는 스크립트 코드로 처리합니다.eval("15+5")→20

3. return문

함수 실행시 데이터를 반환할 때, 강제 종료할 때, 재귀함수를 호출할 때 사용한다.

데이터 반환시 기본형
function 함수명(){
  실행문;
  return 데이터(값);
}
var 변수=함수명(); //함수 호출
강제 종료시 기본형
function 함수명(){
  실행문
  return;
  실행문;
}
함수명(); //return 전까지 함수가 실행된다
재귀 함수 호출시 기본형
funtion 함수명a(){
  실행문;
  함수명a();
}
함수명a();

4. 지역 변수와 전역 변수

지역 변수: 함수 정의문 내에서 var로 정의한 변수. 정의문 내에서만 사용 가능

전역 변수: 함수 정의문 바깥에서 var로 정의한 변수. 스크립트 선언문 내에서 어디든 사용 가능

이름을 붙여 구분하는 이유: 지역 변수는 중복 사용을 피할 수 있으며, 전역 변수는 공용 사용이 가능하다.

함수 정의문 안과 바깥에 같은 변수를 선언하고 바깥의 다른 함수에서 변수를 사용하면: 바깥의 전역 변수가 적용된다.

지역 변수 기본형
function 함수명(){
  var 변수=값;
}
전역 변수 기본형
var 변수;
funtion 함수명(){
  변수=값;
}
