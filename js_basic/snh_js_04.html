객체

객체란

  • Javascript는 객체(Object)기반 언어
  • 객체는 기능 또는 속성을 가지고 있다
  • 객체의 기능: Method
  • 객체의 속성: Property
메서드 사용 기본형
객체.메서드(); //객체의 메서드를 실행
속성 사용 기본형
객체.속성; //객체의 속성값을 가져옴
객체.속성=값; //객체의 속성값을 바꿈

객체의 종류

  • 내장객체: 자바스크립트 엔진에 내장. 크게 날짜date, 수학math, 배열array, 문자string 객체가 있다. 수학, 정규 표현 객체도 있음.
  • 브라우저 객체 모델(BOM): 브라우저에 계층적으로 내장. window, screen, location, history, navigator 객체가 있음
  • 문서 객체 모델(DOM): (X)HTML문서 구조. 하지만 IE8 이하 버전에서 호환성이 떨어짐. 대체가 jQuery 문서 객체 모델.
