객체_문서 객체 모델(DOM:Document Object Model)

브라우저 객체란: 브라우저에 내장된 객체. 계층적 구조로 이루어져 있음

단, DOM은 IE 8 이하 버전과 그 외 하어이폭스, 크롬, 사파리 등의 브라우저 간에 호환성이 떨어진다는 단점이 있음

문서 구조를 보면 html부터 가지처럼 뻗어가는데, 이를 노드(node)라고 한다. 이 노드에는
(x)HTML 태그를 연결하는 요소 노드(Element node),
텍스트를 연결하는 텍스트 노드(Text node),
HTML 태그에 속성을 연결해 주는 속성 노드(Attribute node)가 있다.

  • 선택자
  • 직접 선택자를 사용한 문저 객체 CSS 적용법
  • 인접 관계 선택자로 문서 객체에 스타일 적용하기
  • 노드 타입으로 인접 관계 선택자 문제 해결하기
  • 요소의 속성값 변경, 생성 및 불러오는 방법
  • 문서 객체 이벤트 핸들러 적용하기
  • 폼 요소 선택자

1. 선택자

직접선택자: id, class, Form Name, Element Name

인접 관계 선택자: parentNode, childNodes, firstChild, children, previousSibling, nextSibling

* parentNode, firstChild, previousSibling, nextSibling은 공백 문자 인식 호환성에 문제가 있음

(x)HTML 문단 태그 다음에 자바스크립트가 와야 문단 태그를 선택자를 사용해 선택할 수 있다.

선택자 종류, 부모 자식은 구조도가 기준
구분종류설명
직접 선택자document.getElementById("아이디 명")아이디를 이용해 요소를 선택해 옵니다.
document.getElementsByTagName("요소(태그)명")[순서]요소의 이름을 이용해 요소를 선택해 옵니다. [순서]가 없으면 모두
document.formName.inputName폼 요소에 name 속성을 이용해 요소를 선택합니다.
인접 관계 선택자parentNode선택한 요소의 부모 요소를 선택해 옵니다.
childNodes선택한 요소의 모든 자식 요소를 선택해 옵니다. 선택한 모든 요소가 배열 객체로 저장됩니다.
children선택한 요소의 자식 요소인 태그만 선택해 옵니다. 선택한 모든 요소가 배열 객체로 저장됩니다.
firstChild선택한 요소의 첫 번째 자식 요소만 선택해 옵니다.
previousSibling선택한 요소의 이전에 오는 형제 요소만 선택해 옵니다.
nextSibling선택한 요소의 다음에 오는 형제 요소만 선택해 옵니다.

2. 직접 선택자를 사용한 문서 객체 CSS 적용법

문서 객체 생성 CSS적용 기본형
document.직접 선택자(선택 메서드).style.속성=값;
스타일 속성 사용시 명령어 형태
ex) background-color -> backgroundColor
ex) font-size -> fontSize

3. 인접 관계 선택자로 객체에 스타일 적용하기

DOM에 적용하는 인접 관계 선택자는 IE8이하 버전 외의 브라우저에서는 호환성이 떨어짐

IE 8 이하의 브라우저는 (x)HTML 줄바꿈, 공백을 문자로 인식하지 않지만, 그 외의 브라우저들은 줄바꿈, 공백이 문자로 인식된다.

인접 관계 선택자로 문서 객체 스타일 적용 기본형
document.직접 선택자(선택 메서드).인접 관계 선택자.style.속성=값;

4. 노드 타입으로 인접 관계 선택자 문제 해결하기

노드 타입으로 IE 8 이하 외의 브라우저 호환성 문제를 해결할 수 있음

노드 타입의 종류
종류타입 값
요소(Element) 노드1
속성(Attribute) 노드2
텍스트(Text) 노드3
비표준 인접 관계 선택자 해결 1)
document.getElementById("wrap").children[0].nodeType;
비표준 인접 관계 선택자 해결 2)
var p=document.getElementsByTagName("p")[1];
var nextObj=p.nextSibling;
while(nextObj.nodeType !=1){
 nextObj=nextObj.nextSibling;
}

5. 요소의 속성값 변경, 생성 및 불러오는 방법

문서 객체의 속성, 메서드
종류풀이
요소 선택.속성명요소의 지정한 속성값을 불러옵니다.
요소 선택.속성명="새 값"요소의 지정한 속성값을 새 값으로 변경 또는 생성합니다.
요소 선택.getAttribute("속성");요소의 지정한 속성값을 불러옵니다.
요소 선택.setAttribute("속성","새 값");요소의 지정한 속성값을 새 값으로 변경 또는 생성합니다.
문서 객체의 하위 요소(innerHTML) 속성
종류풀이
요소 선택.innerHTML;선택한 요소의 모든 하위 요소를 문자 데이터로 반환합니다.
요소 선택.innerHTML=새 요소;선택한 요소의 전체 하위 요소를 새 요소로 변경 또는 새롭게 생성합니다.

6. 문서 객체 이벤트 핸들러 적용하기

이벤트: 방문자가 사이트에서 행하는 모든 행위

이벤트 핸들러: 이벤트를 취했을 때 자바스크립트 코드가 실행되는 것

이벤트 핸들러 적용 기본형
요소 선택.이벤트 종류=function(){
 일련의 실행문;
}
이벤트 종류
종류설명
onclick선택한 요소를 마우스로 클릭했을 때 이벤트가 발생합니다.
onmouseover선택한 요소에 마우스를 올렸을 때 이벤트가 발생합니다.
onmouseout선택한 요소에 마우스가 벗어났을 때 이벤트가 발생합니다.
submit선택한 폼에 전송이 일어났을 때 이벤트가 발생합니다.
